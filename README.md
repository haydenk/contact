# contact

Private contact form for reaching Hayden King, hosted via GitHub Pages at [haydenk.github.io/contact](https://haydenk.github.io/contact/).

## Stack

- Plain HTML/CSS/JS — no frameworks
- [Formspree](https://formspree.io) for form submission
- GitHub Pages for hosting

## Usage

Visit the page and fill out the form. Submissions go directly to my inbox.
